## Installation

- Install Node.js & npm
- Install [Rust](https://www.rust-lang.org/tools/install)
- Install [outpost](https://github.com/symil/outpost)
- Run `install.sh`

This script assumes you are running Ubuntu.

## Usage

Run the tests:

```
$ ./test.sh -a
```
